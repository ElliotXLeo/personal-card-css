Página Web Personal versión de Tarjeta, donde el profesional detalla mediante palabras clave un resumen su experiencia e intereses, además cuenta con una presentación donde resume su perfil y metas, finalmente agrega formas de contactarlo.

## Instalación
- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/ElliotXLeo/personal-website-card-rn
  - cd personal-website-card-rn o ir desde el explorador de archivos
  - Abrir el archivo index.html o levantar un servidor con herramientas como Visual Studio Code + Live Server
